# SAP Kernel Upgrade Basics

## Purpose

Kernel upgrade is performed to improve SAP system stability, performance, and security.

## Steps

1. Check current kernel version
2. Download new kernel files
3. Stop SAP system
4. Replace kernel executables
5. Start SAP system
6. Verify kernel version

## Benefits

- Bug fixes
- Performance improvements
- Security enhancements
